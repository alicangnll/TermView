## âœ¨ Features

- **ANSI Rendering**: Automatically parses and renders ANSI escape sequences (colors, bold, etc.).
- **Log Support**: Perfect for viewing `.log` and `.txt` files generated by terminal processes.
- **Natural Integration**: Works as a Custom Editor in VS Code.
- **Theme Friendly**: Designed to look great with VS Code's dark and light themes.

## ğŸš€ How to Use

1. **Right-click** on any `.log` or `.txt` file in your Explorer.
2. Select **Open With...**.
3. Choose **TermView**.
4. Enjoy your colorful logs!

> [!TIP]
> You can set TermView as the default editor for these file types by choosing **Configure default editor for...** in the "Open With..." menu.
